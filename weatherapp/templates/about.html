About This Project

Overview
OTY (Over To You) weather forecaster is a mini project web application built with Python's Django framework (A backend web application for building and maintaining websites) and Bootstrap(A development framework built on HTML, CSS, and JavaScript to facilitate the development of responsive, mobile-first sites and apps) to display weather conditions for a specific city.
This webpage provides two sections.
Section one displays the current weather conditions including the current day and time of the specified city.
Section two displays a graphical representation of annual global mean temperatures from 2004 to 2016 as well as the daily maximun and minimum temperatures for Ghana and Nigeria in a time frame of 2020-01-01 to 2020-01-10.

The Technology Behind
A Technology known as web scraping helps in this work.
As our website was built with Python, we were able to use some of it's web scraping libraries such as BeautifulSoup and Requests.
Requests And BeautifulSoup Libraries
The Requests library is usually used in downloading the contents of a webpage.
The BeautifulSoup library is used to filter the contents which comes in as a source code of the downloaded web page in other to extract specific data within the HTML tags of the source code.
How It Works
A user enters the name of a city in the search bar, the Requests library collects user input and sends an HTTP request to Google's search engine. The response thus the searched result webpage by Google is downloaded by the Requests library which is then stored in a variable to be used by the BeautifulSoup library.
The BeautifulSoup library filters the contents which comes in as a source code of the downloaded web page in other to extract specific data within the HTML tags of the source code.
Extracted data with the help of BeautifulSoup is stored in a Python Dictionary and later rendered to our webpage.
Section Two
Rendered graphs showing real world data on our webpage were made possible by Python's data interactive visualization and manipulation tools known as Bokeh and Pandas respectively.
How It Works
We provided csv datasets and used Pandas to manipulate data from the datasets and stored them to be used by the Bokeh library.
The stored data are then plotted as glyphs (graphical representations) using methods by the Bokeh library and are stored in a variable and later rendered on our webpage.
